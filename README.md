# classic.wgl

## Description
A classic WebGL game engine.

## Develop (git and nix)
1. Clone the repo and install dep:
```bash
git clone <repo-url>
cd classic.wgl
git checkout dev
nix-shell
npm i
```
2. Start the development server:
```bash
npm run dev
```
